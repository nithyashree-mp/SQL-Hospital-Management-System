# Hospital Management System (SQL Project)

## Overview
This project is a Hospital Management System developed using SQL to manage hospital data efficiently, including patients, doctors, departments, appointments, and billing records.
The system demonstrates relational database design and practical implementation of SQL queries in a real-world scenario.

## Features
* Manage patient records
* Store doctor and department details
* Schedule and track appointments
* Maintain billing and payment status
* Perform data analysis using SQL queries

## Database Tables
* Patients
* Doctors
* Departments
* Appointments
* Billing

The tables are connected using Primary Keys and Foreign Keys to maintain data integrity.

## SQL Concepts Used
* CREATE TABLE
* INSERT, UPDATE, DELETE
* SELECT Queries
* WHERE, GROUP BY, ORDER BY
* INNER JOIN, LEFT JOIN
* Aggregate Functions (COUNT, SUM, AVG)
* Constraints (PRIMARY KEY, FOREIGN KEY, NOT NULL)
* SubQueries
* Views

## How to Run
* Open MySQL / SQL Server.
* Create a new database.
* Import or run the provided hospital_management_system.sql file.
* Execute the sample queries.

## Tools Used
* MySQL / SQL Server
* SQL
